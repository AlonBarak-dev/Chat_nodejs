# Chat_nodejs
A localhost chat application demonstrating the use of Web-sockets.

## How to use
- first make sure node.js is install on your machine. <br>
- run:
```
  npm i express socket.io nodemon
```

- start:
```
npm start
```

- Open multiple tabs in your locaL machine and go to:
```
http://localhost:4000
```
